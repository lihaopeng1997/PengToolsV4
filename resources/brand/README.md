# PengTools 品牌标识

## 标识含义

“PengTools”标识不是普通字母 P：它是一条圆润、收拢的工作路径。

- 左上圆点：一次需求、资料或想法的起点。
- 上方闭合路径：整理、归类与形成工作上下文。
- 中段回折：检查、处理和协作动作。
- 向下短线：稳定落地为 SQL、文档、日报或发布结果。

整体在小尺寸时仍可被识别为 P，因此具有产品名记忆点；在大尺寸时又能读成“将杂乱工作收进一个安静工作台”的路径。

## 资源用途

| 文件 | 用途 | 颜色 |
|---|---|---|
| `pengtools-app-mark.svg` | 应用主 Logo、任务栏 ICO 的源图 | 默认静谧办公绿色底 |
| `pengtools-tray-mark.svg` | 系统托盘图标 | `currentColor`，由主题服务染色 |
| `pengtools-floating-mark.svg` | 悬浮框开关按钮 | `currentColor`，由悬浮框表面决定 |

应用窗口、任务栏与安装程序必须使用从 `pengtools-app-mark.svg` 导出的多尺寸 `app.ico`；不要再使用单独的字母 P 或圆形渐变图标。托盘和悬浮框不使用文字，以保证 16–40px 尺寸清晰。

## 尺寸

- ICO：16、20、24、32、40、48、64、128、256px。
- 托盘：16px，Windows 高 DPI 以 20/24px 渲染。
- 悬浮开关：32px 图形放进 44px 点击区域。
- 侧栏品牌：24px 图形放进 36px 品牌底板。
