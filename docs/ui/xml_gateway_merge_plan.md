# XML 格式化 → 网关加解密模块 合并方案

> 状态：**已落地（本轮）**  
> 入口：`panels/gateway_panel.py` · `ui/xml_workspace.py` · `tools/xml_formatter.py`

## 目标

把 XML 能力做成「热门工具」完整度，并**落在网关加解密（GatewayDecodePanel）同一工作面**，避免用户在多个模块间跳转。

## 产品形态（当前）

```
网关加解密
├── 页头（标题 / 副标题 / 离线徽标）
├── 国密参数区（系统 / 环境 / SM4 Key）
└── 工作台 Tab
      ├── 国密解密
      │     ├── 左：密文输入
      │     ├── 右：明文 / JSON 查看器
      │     │       · 一键格式化 JSON · 树节点 · 送入 XML 工具
      │     └── 动作栏：清空 · 复制 · 请求解密 · 响应解密
      └── XML 工具
            ├── 工具条：粘贴 / 清空输入 / 去引号·反转义 / 一键格式化 / 换行 / 复制 / 全部清空
            └── 可拖分栏：输入 | 输出 + 行数字数状态
```

## 功能清单

| 能力 | 状态 | 位置 |
|------|------|------|
| 外层引号清理 | ✅ | `normalize_xml_input` + 工具条「去引号/反转义」 |
| 转义字符处理 | ✅ | 同上 |
| 一键格式化 | ✅ | `format_xml_text` + 工具条主按钮 |
| 错误提示（行列 + 跳行） | ✅ | `show_warning` + 输入区定位 |
| 复制结果 | ✅ | 复制输出（空则复制输入） |
| 清空 | ✅ | 清空输入 / 全部清空 |
| 大文本体验 | ✅ | 等宽、可选换行、横滑、状态条统计 |
| 输入/输出分栏可拖 | ✅ | `QSplitter#xml-splitter` |
| 与解密并列 | ✅ | `module-tabs` 第二页 |
| 解密结果像 XML | ✅ | 状态提示 +「送入 XML 工具」 |

## 不做（边界保持）

- 不引入网络请求 / 在线 schema  
- 不把 XML 工具塞进需求/SQL 面板  
- 不改 `gateway_crypto` 加解密算法  

## 验收

- [x] 粘贴带引号/转义的 XML，一点格式化成功  
- [x] 非法 XML 弹出企业级警告并显示行列  
- [x] 复制 / 清空可用  
- [x] 与解密流程同一面板，无需切换导航  
- [x] 定向单测覆盖逻辑与 UI 挂载
